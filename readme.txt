== Description ==

A easy to use WoW Recruitmet Widget

== Installation ==

1. Upload the `world-of-warcraft-recruitment-widget` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Setup the widget in the widgets settings site in your Wordpress admin area
